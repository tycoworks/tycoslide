# @tycoworks/tycoslide-components

> **Early release (v0.1.0)** — under active development. API may change between minor versions.

Standard component library for tycoslide presentations. Provides text, image, card, table, mermaid diagram, code block, and other components.

Installed automatically as a dependency of theme packages. Install directly if building a custom theme:

```bash
npm install @tycoworks/tycoslide @tycoworks/tycoslide-components
```

See the [full documentation](https://github.com/tycoworks/tycoslide) for details.
